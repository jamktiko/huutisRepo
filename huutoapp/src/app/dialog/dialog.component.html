How to use Votea

Do you want to join a vote?

  • 1. Ask the person who created the room in Votea for the room code.
  • 2. Type the code in the front page codeboxes and press Join.
  • 3. Type in your username or choose a random username and Go vote!

Do you want to make a vote?

  • 1. Select Create Room
  • 2. At the Room Creation View you can enter the question that's voted for, choose the type of vote and add choices to vote from. When you're done setting up the vote, select Create!
  • 3. Your roomcode is displayed at the top of your screen, share this to the people you want to invite to a vote.
  • 4. Type in your username or choose a random username and Go vote!
