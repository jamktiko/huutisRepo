How to use Votea

Do you want to join a vote?

  • 1. Ask the person who made the voting room in Votea for the room code!
  • 2. Type it in on the front page and go wild!

Do you want to make a vote?

  • 1. Select Create Room
  • 2. At the Room Creation View you can enter the question that's voted for, choose the type of vote and add choices to vote from. When you're done setting up the vote, select Create!
  • 3. Choose your username, after that select Go Vote!
  • 4. Vote! Select "Choose for me" if you don't want to choose yourself.
